                   BINARY SEARCH ALGORITHM
*best time complexity-->o(1)   when element finds at first
*Avg case time compl--->o(log(n))
*worst case   --->o(log(n))
*Space complexity --->o(1)   no extra space is required

def fun(arr,target):
    left=0
    right=len(arr)-1
    while left<=right:
        mid=(right+left)//2
        if arr[mid]==target:
            return mid
        elif arr[mid]<target:
            left=mid+1
        else:
            right=mid-1
    return -1

n=int(input())
arr=list(map(int,input().split()))
target=int(input())
print(fun(arr,target))


  
                     BUBBLE SORT
*Bubble sort is used to compare adjacent elements and sort the array
*In each iteration every last element is sorted and process repeats

*best time complexity-->o(n)  when arr is sorted in first iteration
*Avg case time compl--->o(n**2)
*worst case   --->o(n**2)
*Space complexity --->O(1)


def fun(arr):
    for iter in range(len(arr)):
        #len(arr)-1-iter is bcz last element in every iteration is sorted so we no need to visit that
        for i in range(len(arr)-1-iter):
            if arr[i]>arr[i+1]:
                arr[i],arr[i+1]=arr[i+1],arr[i]

    return arr

    int n;
    cin>>n;
    int arr[n],key;
    int maxi=INT_MIN;
    for(int i=0;i<n;i++) cin>>arr[i];
    cin>>key;
    for(int i=0;i<n;i++){
        for(int j=i;j<n;j++){
            int s=0;
            for(int k=i;k<=j;k++){
                s+=arr[k];
            }
            if(s==key) maxi=max(maxi,abs(i-j)+1);
        }

    return 0;
